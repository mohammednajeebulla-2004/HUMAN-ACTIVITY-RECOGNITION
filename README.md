# Human Activity Recognition

A real-time Human Activity Recognition system using pose and gesture detection.

## Features
- Real-time activity detection using webcam
- Posture and gesture recognition
- Built with OpenCV, MediaPipe, and Machine Learning

## How to Run

1. Clone the repo
   git clone https://github.com/mohammednajeebulla-2004/HUMAN-ACTIVITY-RECOGNITION.git

2. Install dependencies
   pip install -r requirements.txt

3. Run the app
   python real_time_app_opencv_v2.py

## Tech Stack
- Python
- OpenCV
- MediaPipe
- Scikit-learn
